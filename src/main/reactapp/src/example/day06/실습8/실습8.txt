조건1 : 제품페이지, 장바구니페이지 구현
-> 제품페이지 MenuPage : 특정 제품을 [담기] 버튼을 클릭하여 전역상태에 저장한다
-> 장바구니페이지 CartPage : 전역상태에 저장된 제품들의 제품명, 수량, 총합계를 출력한다
* 새로고침(F5)을 해도 장바구니는 유지되도록 한다

조건2 : 패키지

    -> store.js
    -> cartSlice.js
