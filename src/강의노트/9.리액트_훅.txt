**** 리액트에서는 훅이 핵심이다 ****
DOM : document object model
    1. HTML문서의 구조를 표현하는 객체
    2. Element : html 마크업들 <div> <input> <p>
    3. Attribute : html 마크업들의 속성 <div style di class src href>
REACT DOM : Virtual document object model
    1. 실제 DOM을 직접 수정하지 않고 가상 DOM을 생성하여 수정/업데이트한다.
    2. Element : 컴포넌트
    3. Attribute : props
[ 생명주기 - life cycle ]
    1. 컴포넌트가 생성되는 시점부터 사라지는 시점까지의 주기
    2. 주기(시점)
        1) mount 마운트(출생) : main.jsx가 컴포넌트를 그려주는 함수.render()
            -> 상의 컴포넌트가 return할 때 하위 컴포넌트가 같이 return되므로 하위컴포넌트들도 출생한다
            -> 컴포넌트가 출생될 때 *1번 실행*되는 함수 : useEffect( ( ) => { } , [ ] )
        2) update 업데이트(인생) : 컴포넌트가 재실행되는 함수
            -> 컴포넌트가 재실행되므로 리렌더링된다 = 화면 새로고침 = 리렌더링 = 컴포넌트 재호출 = 함수 재호출
            -> 컴포넌트가 *업데이트될때마다* 실행 : useEffect( ( ) => { } , [ 의존성배열 ] )
            * 의존성배열이란 ? 의존(참조)배열(목록)으로 특정한 상태( 변수값 - useState )가 변경될 때만
        3) unmount 언마운트(사망) : 컴포넌트가 화면에서 사라질 때 *자동*

[ 훅 ] : 리액트의 컴포넌트(state)와 생명주기 기능에 갈고리를 걸어서 원하는 시점에 정해진 함수를 실행
    1. [ useState ]
    2. [ useEffect ]
